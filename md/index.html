Markdown

What is Markdown?‚Äã

Markdown (MD) is an extremely lightweight markup language created in 2004 by John Gruber. Markdown is lightweight and easy to use, which contributed to it's immense popularity. MD can be used for a number of purposes:

  • Writing documentation
  • Writing books
  • Creating websites
  • Writing notes

Markdown popularity caused it even to be used at sites like Reddit, where you can use MD syntax while writing comments. Another important value of MD is that it is extremely future-proof. Unlike MS Word, it does not lock you into any kind of proprietary format; it allows you to create files in any environment you would like, even online, through use of sites like Dillinger, where you can simultaneously view both your raw md input as well as an already rendered document. An interesting thing about markdown is that due to it being open-source, it can come in variety of flavours. Of course raw md will be rendered correctly every time, but some flavours may not be supported by for example GitHub Pages.

this is an example of markdown syntax.
# this is an example of h1
## this is an example of h2
*this is an example of cursive writing*
